# To-Do List

A simple and interactive to-do list web application with drag-and-drop reordering and local storage.

## Features

- Add, check, and delete tasks
- Drag and drop to reorder tasks
- Tasks persist between sessions using localStorage
- Clean, responsive design

## Getting Started

Simply open `index.html` in your web browser.

## Usage

- **Add a task**: Type in the input field and press Enter or click "Add Item"
- **Complete a task**: Click the checkbox 
- **Delete a task**: Click the red X button
- **Reorder tasks**: Drag and drop tasks to rearrange them

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
